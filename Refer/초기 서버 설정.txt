[초기 서버 설정 가이드 라인]
1. visual studio code 설치
https://code.visualstudio.com/

2. node.js 설치
https://nodejs.org/en

-설치 확인 방법-
-> cmd 창에 node 명령어 입력하고 
Welcome to Node.js v20.12.0.
Type ".help" for more information.
가 뜬다면 정상적으로 설치가 완료 됨.

3. Vs code 폴더 열기
-바탕화면에 아무 폴더나 생성 ex) openSW 폴더 생성
-visual studio code를 실행하고 왼쪽 상단 메뉴바에 
[File]->[Open Folder]->[openSW] 클릭하고 폴더 선택.

4. express 라이브러리 설치 
-상단 메뉴바에 [Terminal] -> [new Terminal] 실행
-명령어 실행창에 npm init 입력 후 엔터
Press ^C at any time to quit.
package name: (d)
version: (1.0.0)
description:
entry point: (index.js)  -----------------------> server.js
git repository:
keywords:
author:
license: (ISC)

엔터를 계속 치다가 entry point 에서 server.js 를 입력 후 계속 엔터

-npm install express 입력하고 설치
-> 왼쪽 파일 목록에 node_modules 패키지가 설치된다면 정상.

5. 서버 열기
- 왼쪽 파일 목록에서 오른쪽 클릭하고 [New file] 선택 후 index.html 입력
- 파일이 생성되면 '!' 치고 엔터 // 안되면 아래 코드를 복사

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>

- 왼쪽 파일 목록에 server.js 파일에 들어가서 다음과 같이 입력

const express = require('express');
const app = express();

app.listen(8080, function(){
    console.log('listening on 8080');
});

- ctrl + s 눌러서 모든 파일 저장
- 터미널에서 npm install -g nodemon 입력 후 설치
- 터미널에 nodemon 입력하면
[nodemon] 3.1.0
[nodemon] to restart at any time, enter `rs`
[nodemon] watching path(s): *.*
[nodemon] watching extensions: js,mjs,cjs,json
[nodemon] starting `node server.js`
listening on 8080
뜨면 서버를 정상적으로 생성.
(서버 종료는 ctrl + c )

- 서버 활성화 확인 방법
-> 인터넷에 localhost:8080으로 접속. 

Cannot GET / 뜨면 정상적으로 열림.
